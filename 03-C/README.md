## Quick Start

- install [GCC] or [Clang] and make sure `gcc` or `clang`
are available in `$PATH`
- `$ CC=gcc make` in case of [GCC]
- `$ CC=clang make` in case of [Clang]


[GCC]: https://gcc.gnu.org/
[Clang]: https://clang.llvm.org/
